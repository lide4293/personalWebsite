Used Git, Java Script, HTML/CSS, React (February 2025 - April 2025)The Vacation Scheduling Tool is designed to efficiently allocate vacation slots to employees based on their requested dates, available openings, and seniority. The system aims to balance employee preferences while ensuring fairness and optimal slot distribution across different locations. The tool takes in three main inputs: a list of available slots by station, employee vacation requests, and employee seniority. The program prioritizes senior employees for popular vacation dates when slots are overbooked. If an employee's requested dates are unavailable, the system will automatically backfill vacation days from mid-November, working backwards, without considering seniority. The goal is to ensure each employee receives 14 vacation days while accommodating holiday peaks and giving employees flexibility for late-year adjustments.

Used VS Code (December 2024) I developed an interactive, 2-player, Lion-King themed game modeled after the Game of Life for children. It includes error handling in order to check if the user input is valid before executing functions. The game takes users through a journey of living as a lion with a visual board that you can see each player move along. Each color tile of the board triggers different events, some randomly generated, that affect the player's Stamina, Wisdom, Strength, and Pride Points. The user with the most Pride points at the end wins the game.
